# Istiak & Eram Wedding Website

This is a React + Vite + Tailwind wedding website.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Vercel

1. Upload this folder to GitHub.
2. Import the GitHub repository in Vercel.
3. Use the default Vite settings.
4. Deploy.

Important: the RSVP form currently uses browser localStorage for testing only. For the real website, connect it to Google Sheets, Airtable, Firebase, Supabase, or another backend.
